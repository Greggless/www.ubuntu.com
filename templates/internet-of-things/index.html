How we define the Internet of Things?

At Canonical we envision IoT devices managed, maintained, and secured the same way the world’s top-tier cloud datacenters already are. Our operating system is the #1 choice for public cloud and OpenStack, so we know a thing or two about enabling large Linux deployments. The Internet’s motley collection of “Things” runs Linux as well, but the contrast is stark. Today’s headlines attest to the chaos; every week we hear about some new IoT vulnerability or giant botnet.

We have the tools to transform unruly hardware platforms into smoothly running device networks. Using Ubuntu Core and snapcraft, you can automate future security updates and create new paths to revenue in the form of add-on apps and remote upgradeability. It’s the wild west out there, let us help you tame your herd!
